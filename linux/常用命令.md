# 常用命令

## 常用
```shell
wc 文件名 -c(统计字节) / -l(统计行数)
date (当前时间)
cal (当前月历)
cat 文件名 (阅读文件内容)
ls -l (列表形式,展示当前文件夹下的文件)
pwd (当前的路径)
touch 文件名 (创建文件夹)
mkdir 文件夹 (创建文件夹)
rm -f 文件名 (强制删除文件)
cd (切换至指定目录)
mv 源文件 目标文件 (移动/覆盖源文件至目标文件)
cp A文件 B文件 (复制A文件给B文件)
echo 变量 (打印变量,如 echo $PWD)
head/tail -n数字 文件名 (读取文件的前/后几行)
```
