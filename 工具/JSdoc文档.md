# JSdoc文档
```java
/*
**一段简单的 JSDoc 注释
*/
@constructor
@class
@type 函数的返回类型
@param {string} author - 书本的作者.
@returns {string|*} 返回当前的书本名称
@overview 对当前代码文件的描述
@copyright 代码的版权信息
@author []代码的作者信息
@version 当前代码的版本
@deprecated 该函数已废弃
@ignore JSDoc会忽略该函数 
```
