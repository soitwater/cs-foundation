# 智力题

## 已知有两个水杯，一个11L一个7L，水可以任意使用，求怎么得到2L 的详细解法
- [参考](https://www.cnblogs.com/j-star/p/7127190.html)
- 思路是一个水桶负责倒入，另一个水桶负责倒出
  * 联立方程组
    - `11x - 7y = 2` && `7x - 11y = 2`
- 代码
  ```js
  let x = 1, y = 1
  for (x = 1; x < 100; x++) {
    for (y = 1; y < 100; y++) {
      if (x * 11 - 7 * y == 2 || x * 7 - 11 * y == 2) {
        console.log('--->', x, y)
      }
    }
  }
   ```
