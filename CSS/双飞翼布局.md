# 双飞翼布局

## 要求
- 三列布局，中间宽度自适应，两边定宽
- 中间栏要在浏览器中优先展示渲染
- 允许任意列的高度最高

## Flexbox实现
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <style type="text/css">
	    div {
	      box-shadow: 0px 0px 3px yellowgreen;
	    }
      .container{
          display: flex;
		  min-width: 1000px;
		  min-height: 400px;
      }
      .main{
        flex-grow: 1;
      }
      .left{
	      order: -1;
		    flex-basis: 200px;
      }
      .right{
		    flex-basis: 300px;
      }
    </style>
</head>
<body>
<div class="container"> 
　　<div class="main">main</div> <!--因为需要优先渲染-->
　　<div class="left">left</div> 
　　<div class="right">right</div> 
</div>
</body>
</html>
```
