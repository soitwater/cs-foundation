# div左右分成两半且为不同颜色

## 举例
```html
<!DOCTYPE html >
<html>
<head>
  <style>
    .half{ 
      height: 50px; 
      width: 0px;  
      border-left: 50px solid red;  
      border-right: 50px solid blue; 
    }
  </style>
</head>
<body>
  <div class='half'></div>
</body>
</html>
```
