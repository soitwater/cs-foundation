# 对语义化的理解

## 理解
- 用正确的标签做正确的事情

## 优点
- 在没有样式CCS情况下也以一种文档格式显示
- 便于`开发人员,屏幕阅读器,爬虫`阅读源码，解析源码

## 常用
即可指标签名，也可以指id以及class的命名
- h1 ~ h6
- p
- ul  ol  li
- dl  dt  dd
- em  strong 
- header  nav  article  section  aside  footer
- th  tbody
- address  
- dfn  code  samp  kbd  var  cite
